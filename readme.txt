== Description ==
This plugin show a graphic counter of hits from user IP. 
Display the counter on widget or blog.
You can set many options for hit and cleanup. 
This plu-in support the de-DE and en_EN language.

== Installation ==
1. Upload Folder `ads_wpsitecount` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the Settings for cleanup IP db entrys when you need.
1. Place Widget ADS-WpSiteCount on your sidebar
1. or add [ads-wpsitecount <args, ...>] in your templates, set params on <args, ...>
